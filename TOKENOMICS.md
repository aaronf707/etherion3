# Etherion (ETR) Tokenomics v1.1

## Core Philosophy
Etherion is designed to be a sustainable, community-driven digital economy. The target value for 1 ETR is **$1.00 USD**, achieved through controlled emission, utility sinks, and long-term scarcity.

## Supply Details
- **Total Supply:** 21,000,000 ETR
- **Decimals:** 6
- **Ticker:** ETR

## Distribution (Emission Pools)
| Pool | Allocation | Purpose |
|------|------------|---------|
| **Mining Rewards** | 60% (12.6M) | PoW Simulation, Hardware Upgrades, Streaks |
| **Engagement** | 30% (6.3M) | Daily Rewards, Referrals, Ad Bonuses |
| **Treasury** | 10% (2.1M) | Ecosystem Development, Liquidity, Mainnet Migration |

## Earning Mechanics (True Data v1.1)
- **Base Hashrate:** 12.0 H/s (Starting CPU)
- **Production Rate:** hashrate * 0.0000025 per second (~0.108 ETR/hr for base users).
- **Sustainability Guard:** Rewards are backed by 75% of app revenue.

## Hardware Multipliers (Base capability)
- **CPU (Base):** 1.0x
- **GTX 1660 GPU:** 2.0x (Requires Level 3 + Data SDK)
- **RTX 4090 GPU:** 4.0x (Requires Level 5 + Data SDK)
- **Small Rig:** 8.0x (Requires Level 10 + Data SDK)
- **Antminer S19:** 12.0x (Requires Level 15 + Data SDK)
- **Enterprise Node:** 20.0x (Requires Level 20 + Data SDK)

## Engagement Bonuses
- **Team Bonus:** +2% per active referral (Cap: 50 members / 2.0x).
- **Streak Bonus:** +1% per day active (Cap: 100 days / 2.0x).
- **Data SDK Opt-In:** +25% direct hashrate boost.

## Utility & Sinks
- **Hardware Depot:** Tokens are spent to upgrade mining equipment.
- **Mainnet Bridge:** Future 1:1 migration to on-chain assets.
- **Node Governance:** Top miners participate in network verification.
