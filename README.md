# Etherion Network (ETR) 🛡️

**Etherion Network** is a next-generation mobile mining node designed to establish a sustainable, community-driven digital economy. Built with Jetpack Compose and powered by real-time data, Etherion allows users to contribute to network security while accumulating ETR tokens.

---

## 🚀 Key Features

- **True Data Dashboard**: Real-time tracking of active miners, node distribution, and projected token value.
- **Hardware Depot**: Spend your earned ETR to upgrade your mining rig from basic CPUs to Enterprise-grade ASIC nodes.
- **Sustainability First**: ETR is backed by 75% of app revenue (AdMob & Anonymous Data SDK), ensuring long-term solvency.
- **Proof of Mining**: A unique 4-hour session cycle that rewards active, human-verified participation.
- **High-Tech UI**: A terminal-inspired interface combined with a modern Material 3 graphical dashboard.
